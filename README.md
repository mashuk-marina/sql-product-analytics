# SQL Product Analytics

## Project Overview

This project demonstrates SQL-based analysis of retail sales data using PostgreSQL.

The goal is to analyze revenue, customer behavior, and product performance through business-oriented SQL queries.

## Business Questions

- What is the total revenue?
- Which products generate the highest revenue?
- Who are the top customers?
- How does revenue change over time?

## Tools

- SQL
- PostgreSQL
- DBeaver
- VS Code
- GitHub

## SQL Skills Demonstrated

- Aggregations
- GROUP BY
- ORDER BY
- LIMIT
- Date Analysis
- Revenue Metrics

## Files

- analytics_queries.sql – SQL queries used for analysis
- OnlineRetail.csv – source dataset (stored locally)

## Key Insights

The project focuses on identifying revenue drivers, customer value, and sales trends that can support business decision-making.
